Cloud Sync - Akses dari PC Lain

Cara Mengakses Video dari PC Lain:
  1. Klik EXPORT (ZIP) untuk mendownload semua video + metadata ke file ZIP
  2. Pindahkan file ZIP ke PC lain (via USB, cloud drive, email, dll)
  3. Di PC lain, buka aplikasi ini dan klik IMPORT (ZIP)
  4. Semua video akan muncul di Library PC lain
  5. Atau bagikan link per video menggunakan tombol Link di setiap card video
Shareable Links: Setiap video memiliki link unik. Salin link dan kirim ke siapa saja. Mereka bisa membuka video jika sudah mengimpor backup atau memiliki file video yang sama.
