Panduan QRIS Premium

Temukan panduan lengkap untuk menggunakan QRIS di berbagai e-wallet dan aplikasi bank

OVO

  1. Buka Aplikasi OVO
  2. Login dengan PIN
  3. Pilih Menu "History"
  4. Klik transaksi QRIS
  5. Temukan Kode RRN
✓

DANA

  1. Buka aplikasi DANA
  2. Klik menu "Riwayat"
  3. Pilih transaksi
  4. Cari "Reff ID"
✓

GoPay

  1. Buka aplikasi GoPay
  2. Ketuk "Lihat semua transaksi"
  3. Pilih transaksi QRIS
  4. Lihat detail & ID transaksi
  5. Gunakan "Bagiin bukti bayar"
✓

ShopeePay

  1. Buka aplikasi Shopee
  2. Masuk ke halaman ShopeePay
  3. Pilih riwayat transaksi
  4. Cari transaksi QRIS
  5. Salin ID transaksi
✓
Tidak ditemukan e-wallet yang sesuai dengan pencarian Anda.

BRI (BRImo)

  1. Buka aplikasi BRImo
  2. Login dengan PIN/sidik jari
  3. Pilih "Transaksi QR" atau "QRIS"
  4. Lihat daftar transaksi
  5. Pilih & simpan bukti transaksi
✓

BCA Mobile

  1. Buka aplikasi BCA Mobile
  2. Masuk dengan password
  3. Pilih menu "QRIS"
  4. Lihat riwayat transaksi
  5. Simpan bukti transaksi
✓

Mandiri Online

  1. Buka aplikasi Mandiri Online
  2. Login dengan username & password
  3. Pilih menu "QRIS"
  4. Cek riwayat transaksi
  5. Download bukti transaksi
✓
Tidak ditemukan bank yang sesuai dengan pencarian Anda.
